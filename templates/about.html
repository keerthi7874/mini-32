About PhishGuard

PhishGuard is a Machine Learning-based system designed to detect phishing URLs. It analyzes multiple features of a URL and predicts whether it is safe or malicious.

🚀 Key Features

  • ✔ Detect phishing URLs using Machine Learning
  • ✔ Real-time URL analysis
  • ✔ Feature-based detection (32 parameters)
  • ✔ Probability-based results
  • ✔ Fast and accurate predictions
